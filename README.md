# BigCommerce Stencil CLI Image

This is a base image designed for working with BigCommerce stencil themes and Github actions, includes:
- NodeJS 14.20.0
- Stencil CLI
- Grunt CLI

## Build

```bash
docker build -t bigcommerce-stencil-cli .
```
